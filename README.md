# calculator-js

Description

A simple and responsive calculator web application built using HTML, CSS, and JavaScript.
It supports light and dark modes, is fully responsive, and works on desktop and mobile devices.

Features

Basic arithmetic operations: addition, subtraction, multiplication, division.

Light and dark mode toggle.

Responsive design for all screen sizes.

Clean and intuitive user interface.

Technologies Used

HTML5 – Structure of the app

CSS3 – Styling, dark/light mode

JavaScript – Calculator logic and interaction

Installation

Clone the repository:

git clone https://github.com/YourUsername/calculator-app.git


Open index.html in your browser.

Usage

Click on the buttons to enter numbers and operations.

Use = to get the result.

Toggle between dark and light mode using the switch button.
